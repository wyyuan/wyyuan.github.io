次梯度法 Bubgradient Optimization Technique

梯度法（爬山法）

选取方向$d$使得$\nabla f(x) d>0$，则$d$就是“上山”方向。

次梯度法核心思想：不断按照一个迭代规则移动$\mu$

  • $x^k$是第$k$次迭代中任意一个解。

在使用这个方法的时候，步长$\theta{k}$的选择较为重要。如果步长太小，算法会卡在一个地方不无法收敛；如果太大，则会漏掉最优解并且可能在两个或多个非优解之间来回。所以通过设置$
\theta{k} \rightarrow 0 $ 和 $ \sum{j=1}^{k} \theta{j} \rightarrow \infty$ 使得算法在这两种极端情况中取一个平衡点。
